Axsync

This module enables portal users / Employee to view and mark their attendance in Odoo. For Customization Contact hunain@axsync.com

Features

- Portal user check-in/check-out.

- Easily filter attendance records directly from the portal.

- Apply filters to view attendance records by date or all records.

- Utilize the search bar for quick access to specific attendance records.

- No additional settings are required.

- This module optimizes efficiency and reduces workload.

Installation

- Copy the portal_attendance module to the addons folder.

- Install the module as you would any other Odoo module.
